# Integrity 데모 앱

`react-native-nitro-device-integrity`의 데모 앱입니다. 하드웨어 기반 디바이스 어테스테이션(Android Play Integrity, iOS App Attest)을 시연합니다.

## 앱 실행 방법

### 저장소 루트에서

```bash
yarn integrity-demo ios      # iOS
yarn integrity-demo android  # Android
```

### 이 디렉토리에서

```bash
cd example/integrity-demo
yarn ios      # iOS
yarn android  # Android
```
